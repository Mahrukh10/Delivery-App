# Delivery-App
=======
This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Choices

The project is made in Next js and Tailwind CSS. Components have been used in the project to make it easier to implements changes (if any).

## Improvements 

More animations or styling would have been implemented if more time was available.
